My Image Search Abstraction Layer

A full stack JavaScript app that allows you to search for images :

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL
  • I can get a list of the most recently submitted search strings

Example search usage:

https://salty-fortress-39775.herokuapp.com/search/cats
https://salty-fortress-39775.herokuapp.com/search/lolcats%20funny?offset=10
